# 猫咪救助站 - 单词游戏

一个有趣的H5单词学习游戏，通过救助猫咪来学习英语单词。

## 游戏说明

### 游戏目标
- 通过移动救助车，收集当前挑战单词所需的字母
- 每个猫咪身上携带一个字母
- 当救助车碰到带有正确字母的猫咪时，会收集该字母并得分
- 收集完当前单词的所有字母后，会自动进入下一个单词

### 游戏操作
- **移动救助车**: 触摸屏幕（移动端）或点击并拖动鼠标（桌面端）来移动救助车
- **重新开始**: 点击"重新开始"按钮重置游戏
- **下一个单词**: 点击"下一个单词"按钮跳过当前单词

### 游戏特色
- 🎮 流畅的触摸控制
- 🐱 97张可爱的猫咪图片随机显示
- 📚 包含大量英语单词（从words_data.json加载）
- 🎯 实时显示需要收集的字母
- 📊 分数系统追踪学习进度

## 使用方法

1. 确保项目目录结构正确：
   - `cats_game/` - 游戏文件目录
   - `cats/` - 猫咪图片目录（包含cat_01.png到cat_97.png）
   - `words_data.json` - 单词数据文件

2. 在浏览器中打开 `index.html` 文件

3. 建议使用本地服务器运行（避免跨域问题）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx http-server
   ```

4. 在浏览器中访问 `http://localhost:8000/cats_game/`

## 技术说明

- 使用HTML5 Canvas进行游戏渲染
- 支持触摸事件和鼠标事件
- 响应式设计，适配移动端和桌面端
- 使用requestAnimationFrame实现流畅动画

## 游戏机制

1. **单词选择**: 从words_data.json中按顺序加载单词
2. **猫咪生成**: 每2秒生成一只新猫咪，最多同时存在8只
3. **字母分配**: 70%概率生成当前需要的字母，30%概率生成随机字母
4. **碰撞检测**: 使用圆形碰撞检测算法
5. **字母收集**: 收集到正确字母后，对应字母会高亮显示
